### 2. State, Strategy, Decorator Patterns

Thư mục này chứa 3 bài tập áp dụng các Design Pattern:
- **Problem1_OrderManagement**: Hệ thống quản lý đơn hàng
- **Problem2_TaxCalculation**: Hệ thống tính thuế
- **Problem3_PaymentSystem**: Hệ thống thanh toán

## Cách chạy

Biên dịch và chạy từng bài tập:

```bash
# Problem 1
cd Problem1_OrderManagement
javac *.java
java Main

# Problem 2
cd Problem2_TaxCalculation
javac *.java
java Main

# Problem 3
cd Problem3_PaymentSystem
javac *.java
java Main
```
